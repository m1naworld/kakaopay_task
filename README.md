
## 카카오 페이 사전 과제 🏃

### 선택 기술 
- Java
- Springboot
- JPA
- Mysql

<br/>

### 필수 요구 사항
- 아래 API들을 구현합니다. 
  1) 커피메뉴목록조회API
  2) 포인트 충전하기 API
  3) 커피 주문, 결제하기 API
  4) 인기메뉴 목록 조회 API
  
- 작성하신 어플리케이션이 다수의 서버에 다수의 인스턴스로 동작하더라도 기능에 문제가 없도록 설계되어야 합니다.
- 각 기능 및 제약사항에 대한 단위테스트를 반드시 작성합니다.
- 동시성 이슈를 고려하여 구현합니다.
- 데이터 일관성을 고려하여 구현합니다.
